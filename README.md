# Установщик для Ubuntu с автоматической настройкой зависимостей

## Описание

Данный установщик предназначен для автоматической подготовки сервера 
Он выполняет проверку системных зависимостей, устанавливает недостающие пакеты, выпускает SSL-сертификаты t и настраивает веб-сервер

Установщик рассчитан на чистую или минимально настроенную систему.

---

## Возможности

- Проверка наличия необходимых пакетов  
- Автоматическая установка зависимостей  
- Создание виртуального хоста  
- Настройка автоматического продления сертификата  
- Перезапуск служб после конфигурации
